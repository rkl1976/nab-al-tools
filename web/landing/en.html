Features

AI Chat

Translate through conversation with AI. Ask about terminology, request alternative phrasings, or let the AI translate entire sections.

Batch Translation

Translate hundreds of texts automatically with Azure OpenAI. Get high-quality translations that respect BC terminology.

Review Workflow

Review and approve translations one by one. See source and target text side by side with AL code context.

Search

Find texts with regex and keyword search. Filter by translation status, notes, and more.

Get started

Installation

Ready in minutes. Download, extract, and run.

1

Download

Get the latest version from GitHub Releases. Choose NabXliffApp-win-x64.zip.

2

Prerequisites

Node.js 20+ required for the MCP server.
WebView2 Runtime must be installed — download here. Already included in the latest versions of Windows 10/11.

3

Extract and run

Extract the ZIP file to any folder and start NabXliffApp.exe. No installation required.

4

Configure Azure OpenAI (optional for AI features)

Open Settings in the app and fill in your Azure OpenAI details. Alternatively, edit appsettings.json directly: 

{
  "AzureOpenAI": {
    "Endpoint":       "https://your-instance.openai.azure.com/",
    "DeploymentName": "gpt-4o",
    "ApiKey":          "your-api-key"
  },
  "McpServer": {
    "Command":  "node",
    "Args":     "sti/til/nab-al-tools/dist/mcp/server.js"
  }
}
